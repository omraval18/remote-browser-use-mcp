Best default: middle dot

It is a one-character swap from the colon, preserves alignment, and reads as a quiet event marker. This is the safest implementation path.

Best hierarchy: vertical rail

Use this only if subagent output becomes a true nested thread. It looks intentional, but it also makes the transcript feel more like a log tree.

Avoid: chevron

The chevron fights the user prompt's greater-than marker and makes activity rows feel interactive or prompt-like when they are not.
