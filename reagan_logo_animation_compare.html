2. Animation modes


All six panels render with quadrant blocks. Difference is the rotation driver.



  
1. Coin flip
x-axis spin. Dramatic вҖ” boot splash.

  
2. Y-axis spin
Globe rotation. Both rings stay visible. Persistent.

  
3. Tumble
Combined xy. Visually rich, distracting.

  
4. Phase shift
Each ring tumbles on a different axis. Idle.

  
5. Wobble
Gentle Вұ25В° oscillation. Calmest option.

  

    
6. Click to throw

    
Click anywhere on the panel. Decays to idle drift.

  Responsive rules baked into every variant:
  the row is marker (1) + command (max-8) + 2-space gap + description (flex) + keyhint (max-4). As the terminal narrows: drop keyhints first, then ellipsize descriptions to a min(12) floor, then drop descriptions entirely. The command column is the floor вҖ” it never gets touched. The bordered frame in B/C subtracts 4 cells of chrome (border + 1-space padding each side) from the width budget before rows are laid out.
