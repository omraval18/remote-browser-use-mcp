Terminal Transcript Colorways

Same transcript sample, same semantic roles. The options focus on muted and pastel palettes that are already familiar in terminal and editor themes.

These are intentionally different families now. The terminal background stays fixed; only the semantic role colors change, so each option should have a distinct temperature and accent system.
